# Multicooker-based-filament-dryer
Multicooker Filament Dryer Controller  
Прошивка для контроллера самодельной сушилки филамента (пластика), построенной на базе корпуса и нагревательных элементов мультиварки. Управление тремя зонами нагрева и вентилятором.

Основные возможности

    Управление 2-мя нагревателями: Раздельное управление боковым (Side), верхним (Top) ТЭНами. 
    (Нижний не используется, слишком мощный. Боковой+верхний догоняют температуру до 90 градусов, только боковой 84 гразуса)
 
    Датчик температуры: Работа с прецизионным цифровым датчиком DS18B20 через библиотеку DS18B20_INT.

    Индикация: 2-разрядный семисегментный индикатор (драйвер MAX7219) отображает текущую температуру или статус зон (HI/LO/--).

    Статусный LED: Светодиод работы (Active-Low), загорается при активации любого нагревателя.

    Безопасный опрос датчика: Асинхронное чтение температуры (запрос -> ожидание в цикле -> чтение) без блокировки основного потока.

Назначение пинов (Pinout)
Устройство	Пин Arduino	Примечание
DS18B20	D3	Датчик температуры
MAX7219 DIN	A4	Дисплей
MAX7219 CS	A5	Дисплей
MAX7219 CLK	A6	Дисплей
Button PLUS	D5	Вкл/выкл боковой нагрев (Side)
Button MINUS	D6	Вкл/выкл верхний нагрев (Top)
Button STOP	D7	Полная остановка всех систем
Button START	D4	Просмотр статуса (длинное нажатие)
Heater SIDE	D10	Реле бокового ТЭНа
Heater TOP	D9	Реле верхнего ТЭНа
Heater BOTTOM	D8	Реле нижнего ТЭНа
Status LED	D2	Инвертированный (LOW = ON)
Логика управления
Кнопки:

    PLUS (+): Переключает состояние Бокового нагревателя. На дисплее кратковременно отображается SH (Side Heater), затем UP (Включен) или OF (Выключен).

    MINUS (-): Переключает состояние Верхнего нагревателя. На дисплее отображается UH (Upper Heater), затем UP или OF.

    STOP: Немедленно выключает все нагреватели. На дисплей выводится OF -> AL (All Off).

    START (Длинное нажатие): Информационный режим. Поочередно показывает активные зоны нагрева (SH, UH) и итоговый статус UP.

Дисплей:

    Отображает текущую температуру в градусах Цельсия.

    Если температура выше 99°C — показывает HI.

    Если температура ниже 0°C — показывает LO.

Используемые библиотеки

    LedControl — для работы с драйвером дисплея MAX7219.

    OneButton — для обработки нажатий кнопок (антидребезг, клики, удержание).

    OneWire & DS18B20_INT — для работы с термометром.

Установка

    Склонируйте репозиторий.

    Установите указанные выше библиотеки через Library Manager в Arduino IDE.

    Проверьте схему подключения согласно таблице пинов.

    Загрузите код в вашу Arduino (рекомендуется Nano или Uno).
